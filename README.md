# Asistente Legal Laboral Peru

## Descripción del Proyecto
El Asistente Legal Laboral Peru es una aplicación diseñada para facilitar la gestión de trámites legales laborales en Perú. Proporciona herramientas y recursos para abogados y empleados que necesitan orientación sobre derechos laborales y procedimientos legales.

## Características Principales
- **Asesoría Legal:** Información sobre derechos laborales, leyes y procedimientos en Perú.
- **Guías paso a paso:** Procedimientos documentados para la realización de trámites legales.
- **Calculadoras de indemnizaciones:** Herramientas para calcular indemnizaciones por despido.
- **Actualizaciones legislativas:** Información sobre cambios en la legislación laboral peruana.

## Instalación
Para instalar el Asistente Legal Laboral Peru, siga estos pasos:
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/victorLopQ/asistente-legal-laboral-peru.git
   ```
2. Navegar al directorio del proyecto:
   ```bash
   cd asistente-legal-laboral-peru
   ```
3. Instalar las dependencias (si corresponde):
   ```bash
   npm install
   ```

## Uso
Para utilizar la aplicación, ejecute el siguiente comando:
```bash
npm start
```
Luego abra su navegador y dirígete a `http://localhost:3000`.
